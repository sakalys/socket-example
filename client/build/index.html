Socket.io chat demo

The following is a chat tool that I built using socket.io to showcase how multiple chat clients can interoperate. Each box (frame) has it's own connection to the chat server. You can add more frames by pressing on Add frame, and remove frames by pressing on the "X" on top of each of them.

The source for this demo is publicly visible on Github — https://github.com/sakalys/socket-example.

Each frame connects to the #public channel automatically. There is an additional #private channel, but to be able to post to it, you must first identify yourself by entering the command /private {pass-phrase} (the pass-phrase in this demo is the word secret. The pass-phrase is checked by the server and if it is correct, then the frame gets access to the #private channel as well. Messages sent to the #private channel will only be visible to other members that are in the #private channel.

Available commands:

  • /private {pass-phrase} — access to admin channel (the pass-phrase is secret)
  • /count — members in channel
  • /quit — close the frame
